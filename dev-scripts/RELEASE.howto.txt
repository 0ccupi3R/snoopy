################################################################################
###
### How to release a new version of Snoopy Logger
###
################################################################################

Steps:
1. update ChangeLog, commit
2. run autoscan, compare configure.scan to configure.ac,
    port required changes to .ac and run autoreconf
2. bump version in configure.ac, run ./autogen.sh, commit
3. create git tag 'snoopy-X.Y.Z'
4. cd to the repo root and run "./dev-scripts/make-release.sh snoopy-x.y.z"
